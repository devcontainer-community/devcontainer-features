# SWC

## Project

- [SWC](https://swc.rs)

## Description

A super-fast JavaScript and TypeScript compiler written in Rust. `swc` can be used as a replacement for Babel and is used by tools like Next.js and Parcel for ultra-fast transpilation.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/swc-project/swc/releases) and placed in `/usr/local/bin`.
