# Jujutsu (`jj`)

## Project

- [Jujutsu (`jj`)](https://jj-vcs.dev)

## Description

A Git-compatible version control system with a clean and powerful model. `jj` operates on a working-copy-as-a-commit paradigm, making rebasing and conflict resolution simpler, while remaining fully compatible with existing Git repositories.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/jj-vcs/jj/releases) and placed in `/usr/local/bin`.
