# Biome

## Project

- [Biome](https://biomejs.dev)

## Description

A fast formatter and linter for JavaScript, TypeScript, JSX, TSX, JSON, CSS, and GraphQL. Biome is a single tool that replaces Prettier and ESLint, providing fast performance through its Rust implementation.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/biomejs/biome/releases) and placed in `/usr/local/bin`.
