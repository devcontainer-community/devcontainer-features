# pkgx

## Project

- [pkgx](https://pkgx.sh)

## Description

A blazingly fast package runner built on top of the Pkgs package manager. `pkgx` runs any package from the ecosystem without installing it first, similar to `npx` but for any language or tool.

## Installation Method

Installed via the official install script: `curl -fsSL https://pkgx.sh | sh`.
