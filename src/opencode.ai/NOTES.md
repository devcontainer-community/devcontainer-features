# opencode

## Project

- [opencode](https://opencode.ai)

## Description

An AI coding assistant that runs in the terminal. `opencode` integrates AI capabilities into the command-line workflow for code generation, explanation, and refactoring.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/anomalyco/opencode/releases) and placed in `/usr/local/bin`.
