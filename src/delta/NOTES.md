# delta

## Project

- [delta](https://github.com/dandavison/delta)

## Description

A syntax-highlighting pager for `git`, `diff`, `grep`, and `blame` output. Delta improves the terminal diff experience with side-by-side view, line numbers, syntax highlighting, and easy configuration via `~/.gitconfig`.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/dandavison/delta/releases) and placed in `/usr/local/bin`.
