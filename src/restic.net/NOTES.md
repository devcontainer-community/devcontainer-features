# restic

## Project

- [restic](https://restic.net)

## Description

A modern backup program that is fast, efficient, and secure. `restic` supports multiple backends (local, S3, B2, SFTP, etc.), deduplicates data, and encrypts all backups by default.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/restic/restic/releases) and placed in `/usr/local/bin`.
