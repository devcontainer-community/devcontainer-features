# Webi (`webi`)

## Project

- [Webi (`webi`)](https://webinstall.dev)

## Description

A simple and fast package installer for developer tools. `webi` installs binaries directly to `~/.local/bin` and manages them in a cross-platform-friendly way, with no sudo required.

## Installation Method

Installed to the remote user's environment via the official install script: `curl -sS https://webi.sh/webi | sh`.
