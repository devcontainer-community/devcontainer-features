# Helix

## Project

- [Helix](https://helix-editor.com)

## Description

A post-modern modal text editor inspired by Neovim and Kakoune. `hx` (Helix) features multiple selections, built-in language server protocol (LSP) support, and tree-sitter-based syntax highlighting, with a focus on composability and discoverability.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/helix-editor/helix/releases) and installed to `/usr/local/lib/helix` with a symlink at `/usr/local/bin/hx`. The runtime files are included alongside the binary.
