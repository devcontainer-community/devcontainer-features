# onecli.sh-cli

## Project

- [onecli.sh](https://onecli.sh)

## Description

CLI for [OneCLI](https://onecli.sh) — manage agents, secrets, rules, and configuration from the terminal. `onecli` lets you authenticate, list and create agents, and manage secrets and policy rules via a simple command-line interface.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/onecli/onecli-cli/releases) and placed in `/usr/local/bin`.
