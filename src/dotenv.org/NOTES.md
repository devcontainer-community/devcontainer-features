# dotenv.org

## Project

- [dotenvx](https://dotenv.org)

## Description

`dotenvx` is a better dotenv — load env vars from `.env` files, support multiple environments, and encrypt secrets. It extends the original dotenv with multi-environment support and encrypted `.env` files.

## Installation Method

Installed as a global package via [Bun](https://bun.sh) (`bun install -g @dotenvx/dotenvx`). Bun is automatically installed if not already present.
