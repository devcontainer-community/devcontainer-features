# dolthub.com

## Project

- [dolt](https://github.com/dolthub/dolt)

## Description

Dolt is a SQL database with Git-style versioning. `dolt` lets you branch, merge, commit, and diff your data just like you version your code, with full MySQL compatibility.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/dolthub/dolt/releases) and placed in `/usr/local/bin`.
