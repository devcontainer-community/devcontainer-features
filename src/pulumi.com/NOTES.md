# Pulumi

## Project

- [Pulumi](https://pulumi.com)

## Description

An infrastructure-as-code platform that allows you to define cloud resources using familiar programming languages (TypeScript, Python, Go, C#, Java, and YAML). Pulumi supports all major cloud providers.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/pulumi/pulumi/releases) and placed in `/usr/local/bin`.
