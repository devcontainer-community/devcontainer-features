# Mosh

## Project

- [Mosh](https://mosh.org)

## Description

A mobile shell that acts as a replacement for interactive SSH sessions. Mosh is resilient to network interruptions and roaming, uses UDP for communication, and maintains responsiveness even on high-latency or intermittently connected networks.

## Installation Method

Installed via the system APT package manager (`apt-get install mosh`).
