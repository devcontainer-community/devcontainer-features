# Cloudflare WARP CLI

## Project

- [Cloudflare WARP CLI](https://developers.cloudflare.com/warp-client/)

## Description

A command-line interface for Cloudflare WARP, a privacy-focused VPN/proxy service. `warp-cli` allows connecting to the Cloudflare network to improve performance and privacy for network traffic.

## Installation Method

Installed via APT from the official [Cloudflare package repository](https://pkg.cloudflareclient.com/) (`cloudflare-warp` package), with the Cloudflare GPG key added to the system keyring.
