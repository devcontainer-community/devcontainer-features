# diffity.com

## Project

- [diffity](https://diffity.com)

## Description

An agent-agnostic, GitHub-style diff viewer and code review tool. `diffity` works with Claude Code, Cursor, Codex, and any AI coding agent — run it inside any git repo to open a syntax-highlighted diff in the browser.

## Installation Method

Installed as a global npm package via `npm install -g diffity`. Node.js and npm are installed as prerequisites via APT.
