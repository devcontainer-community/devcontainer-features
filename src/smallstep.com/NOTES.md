# step CLI

## Project

- [step CLI](https://smallstep.com/cli/)

## Description

A general-purpose cryptography and certificate management toolkit. `step` makes working with X.509 certificates, JWT tokens, ACME, OAuth/OIDC, and other cryptographic primitives approachable from the command line.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/smallstep/cli/releases) and placed in `/usr/local/bin`.
