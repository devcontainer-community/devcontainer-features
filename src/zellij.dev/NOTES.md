# Zellij

## Project

- [Zellij](https://zellij.dev)

## Description

A terminal workspace and multiplexer. Zellij provides panes, tabs, and layouts with a built-in plugin system and floating pane support, making it a modern alternative to `tmux` and `screen`.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/zellij-org/zellij/releases) and placed in `/usr/local/bin`.
