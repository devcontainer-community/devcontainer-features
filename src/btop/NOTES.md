# btop

## Project

- [btop](https://github.com/aristocratos/btop)

## Description

A resource monitor that shows usage and stats for processor, memory, disks, network, and processes. `btop` provides a beautiful and responsive TUI with mouse support and extensive configuration options.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/aristocratos/btop/releases) and placed in `/usr/local/bin`.
