# critique

## Project

- [critique](https://critique.work)

## Description

An AI-powered code review tool. `critique` analyses code changes and provides automated review feedback directly from the terminal.

## Installation Method

Installed as a global package via [Bun](https://bun.sh) (`bun install -g critique`). Bun is automatically installed if not already present.
