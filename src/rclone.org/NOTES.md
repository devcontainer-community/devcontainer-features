# rclone

## Project

- [rclone](https://rclone.org)

## Description

A command-line program for managing files on cloud storage. `rclone` supports over 70 cloud storage providers including S3, Google Drive, Azure Blob Storage, and Dropbox, providing sync, copy, mount, and serve operations.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/rclone/rclone/releases) and placed in `/usr/local/bin`.
