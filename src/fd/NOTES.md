# fd

## Project

- [fd](https://github.com/sharkdp/fd)

## Description

A simple, fast, and user-friendly alternative to the Unix `find` command. `fd` features an intuitive syntax, colorized output, smart case-sensitivity, and `.gitignore`-awareness.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/sharkdp/fd/releases) and placed in `/usr/local/bin`.
