# lnav.org

## Project

- [lnav](https://github.com/tstack/lnav)

## Description

`lnav` is an advanced log file viewer for the terminal. It provides automatic log format detection, syntax highlighting, filtering, and search across multiple log files simultaneously.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/tstack/lnav/releases) and placed in `/usr/local/bin`.
