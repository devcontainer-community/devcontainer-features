# sxyazi/yazi

## Project

- [yazi](https://github.com/sxyazi/yazi)

## Description

Blazing fast terminal file manager written in Rust, based on async I/O. `yazi` provides a full-featured file manager with image preview, multi-tab support, fuzzy find, and plugin support.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/sxyazi/yazi/releases) and placed in `/usr/local/bin`.
