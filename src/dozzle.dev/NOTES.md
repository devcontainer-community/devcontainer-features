# Dozzle

## Project

- [Dozzle](https://dozzle.dev)

## Description

A real-time log viewer for Docker containers. `dozzle` provides a lightweight web-based interface to browse and follow container logs without storing them.

## Installation Method

The `dozzle` binary is extracted directly from the [official Docker Hub image](https://hub.docker.com/r/amir20/dozzle) (`amir20/dozzle`) and placed in `/usr/local/bin`.
