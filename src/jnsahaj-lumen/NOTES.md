# lumen

## Project

- [lumen](https://github.com/jnsahaj/lumen)

## Description

An AI-powered git diff viewer and commit message generator. `lumen` integrates with your git workflow to provide contextual explanations of diffs and automatically draft commit messages.

## Installation Method

Installed via [Cargo](https://doc.rust-lang.org/cargo/) from [crates.io](https://crates.io/crates/lumen). Rust is automatically installed using `rustup` if not already present.
