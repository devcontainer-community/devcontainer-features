# Bun

## Project

- [Bun](https://bun.sh)

## Description

An all-in-one JavaScript runtime and toolkit. Bun includes a fast JavaScript/TypeScript runtime, bundler, test runner, and Node.js-compatible package manager — designed as a drop-in replacement for Node.js.

## Installation Method

Installed to the remote user's home directory via the official install script: `curl -fsSL https://bun.sh/install | bash`.
