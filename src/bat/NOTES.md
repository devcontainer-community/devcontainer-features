# bat

## Project

- [bat](https://github.com/sharkdp/bat)

## Description

A `cat` clone with syntax highlighting and Git integration. `bat` supports a large number of programming and markup languages and integrates with other tools such as `fzf`, `ripgrep`, and `delta`.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/sharkdp/bat/releases) and placed in `/usr/local/bin`.
