# memvid

## Project

- [memvid](https://memvid.com)

## Description

A video-based memory system for AI. `memvid` allows encoding large amounts of text or knowledge into video files and then performing fast semantic search over that stored memory.

## Installation Method

Installed as a global npm package via `npm install -g memvid-cli`. Node.js and npm are installed as prerequisites via APT.
