# obsidian-cli

## Project

- [obsidian-cli](https://github.com/Yakitrak/obsidian-cli)

## Description

A command-line tool for interacting with Obsidian vaults from the terminal. `obsidian-cli` allows creating, reading, updating, searching, and deleting notes in your Obsidian vault without opening the app.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/Yakitrak/obsidian-cli/releases) and placed in `/usr/local/bin`.
