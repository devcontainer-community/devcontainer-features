## Description

An AI-powered programming assistant. `pi` provides an interactive terminal interface for discussing code, generating solutions, and getting contextual programming help.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/badlogic/pi-mono/releases) and placed in `/usr/local/bin`.
