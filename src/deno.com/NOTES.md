# Deno

## Project

- [Deno](https://deno.com)

## Description

A secure runtime for JavaScript, TypeScript, and WebAssembly. Deno is built on V8 and Rust, supports TypeScript out of the box, and provides a secure-by-default sandbox with explicit permission flags.

## Installation Method

Installed system-wide (to `/usr/local`) via the official Deno install script: `curl -fsSL https://deno.land/install.sh | sh`.
