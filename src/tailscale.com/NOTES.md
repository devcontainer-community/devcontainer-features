# Tailscale

## Project

- [Tailscale](https://tailscale.com)

## Description

A zero-config VPN mesh network built on WireGuard. Tailscale creates a private network between your devices without complex firewall or routing configuration, and works across NAT boundaries.

## Installation Method

Installed via the official Tailscale install script: `curl -fsSL https://tailscale.com/install.sh | sh`.
