# chezmoi

## Project

- [chezmoi](https://chezmoi.io)

## Description

A dotfiles manager that helps you manage your personal configuration files across multiple machines. chezmoi supports templates, secret managers, and keeps track of differences between local files and the desired state.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/twpayne/chezmoi/releases) and placed in `/usr/local/bin`.
