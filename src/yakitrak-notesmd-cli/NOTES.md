# notesmd-cli

## Project

- [notesmd-cli](https://github.com/Yakitrak/notesmd-cli)

## Description

A command-line tool for interacting with Obsidian vaults from the terminal. `notesmd-cli` allows creating, reading, updating, searching, and deleting notes in your Obsidian vault without opening the app.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/Yakitrak/notesmd-cli/releases) and placed in `/usr/local/bin`.
