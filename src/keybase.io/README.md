## Description

A key directory and encrypted communication platform. The `keybase` CLI provides cryptographic identity verification, end-to-end encrypted chat and file sharing, and PGP key management tied to public social identities.

## Installation Method

The `keybase` CLI binary is extracted directly from the official `.deb` package (downloaded from `https://prerelease.keybase.io/`) without installing the full Electron GUI application, keeping the container footprint small.
