# Neovim

## Project

- [Neovim](https://neovim.io)

## Description

A hyperextensible Vim-based text editor. Neovim refactors Vim to enable a first-class plugin system, an embedded Lua scripting engine, and built-in LSP client, making it a powerful and highly configurable editor for developers.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/neovim/neovim/releases) and placed in `/usr/local/bin`.
