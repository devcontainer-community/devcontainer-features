# hermes-agent.nousresearch.com

## Project

- [Hermes Agent](https://github.com/NousResearch/hermes-agent)

## Description

The self-improving AI agent built by [Nous Research](https://nousresearch.com). Hermes has a built-in learning loop — it creates skills from experience, improves them during use, and builds a deepening model of who you are across sessions. Supports any model via OpenRouter, OpenAI, Anthropic, and more.

## Installation Method

Installed to the remote user's home directory via the official install script: `curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash -s -- --skip-setup`.
