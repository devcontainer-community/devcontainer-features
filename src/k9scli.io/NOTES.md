# K9s

## Project

- [K9s](https://k9scli.io)

## Description

A terminal-based UI for interacting with Kubernetes clusters. `k9s` provides a real-time view of Kubernetes resources, allows navigating and managing workloads, and features a powerful command mode.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/derailed/k9s/releases) and placed in `/usr/local/bin`.
