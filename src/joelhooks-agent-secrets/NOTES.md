# agent-secrets (`secrets`)

## Project

- [agent-secrets (`secrets`)](https://github.com/joelhooks/agent-secrets)

## Description

A tool for managing secrets in AI agent workflows. `secrets` provides a simple interface for storing and retrieving secrets used by automated agents.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/joelhooks/agent-secrets/releases) and placed in `/usr/local/bin`.
