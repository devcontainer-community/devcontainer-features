# fathyb/carbonyl

[carbonyl](https://github.com/fathyb/carbonyl) is a Chromium-based browser built to run in a terminal. It supports most web features including JavaScript, CSS, and media.

Carbonyl is downloaded from [GitHub Releases](https://github.com/fathyb/carbonyl/releases) and installed to `/usr/local/lib/carbonyl/`, with the `carbonyl` binary symlinked to `/usr/local/bin/carbonyl`.
