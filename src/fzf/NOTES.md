# fzf

## Project

- [fzf](https://github.com/junegunn/fzf)

## Description

A general-purpose command-line fuzzy finder. `fzf` can be used as an interactive filter for any list — files, git history, processes, hostnames, commands, etc. — and integrates with shells, Vim/Neovim, and other tools.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/junegunn/fzf/releases) and placed in `/usr/local/bin`.
