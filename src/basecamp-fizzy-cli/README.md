## Description

`fizzy` is a command-line interface for [Fizzy](https://fizzy.do), a project management tool by Basecamp. Manage boards, cards, comments, and more from your terminal or through AI agents. It works standalone or with any AI agent (Claude, Codex, Copilot, Gemini) and returns structured JSON output with breadcrumbs for easy navigation.
