# lazygit

## Project

- [lazygit](https://github.com/jesseduffield/lazygit)

## Description

A simple terminal UI for git commands. `lazygit` allows staging files, viewing diffs, committing, rebasing, resolving merge conflicts, and browsing commit history — all from an intuitive keyboard-driven interface.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/jesseduffield/lazygit/releases) and placed in `/usr/local/bin`.
