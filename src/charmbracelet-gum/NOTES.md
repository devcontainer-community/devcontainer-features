# Gum

## Project

- [Gum](https://github.com/charmbracelet/gum)

## Description

A tool for making glamorous shell scripts. `gum` provides a set of interactive UI components — input prompts, fuzzy finders, spinners, and more — that can be composed directly in shell scripts.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/charmbracelet/gum/releases) and placed in `/usr/local/bin`.
