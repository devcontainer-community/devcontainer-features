# turso.tech

## Project

- [turso](https://github.com/tursodatabase/turso)

## Description

Turso is an in-process SQL database written in Rust, compatible with SQLite. The `tursodb` CLI provides an interactive shell for executing SQL statements, as well as a built-in Model Context Protocol (MCP) server for AI assistant integration.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/tursodatabase/turso/releases) and placed in `/usr/local/bin`.
