# Starship

## Project

- [Starship](https://starship.rs)

## Description

A minimal, blazing-fast, and infinitely customizable prompt for any shell. Starship shows information about the current directory, git branch, programming language versions, and much more, with sensible defaults.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/starship/starship/releases) and placed in `/usr/local/bin`.
