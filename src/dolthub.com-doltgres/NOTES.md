# dolthub.com/doltgres

## Project

- [DoltgreSQL](https://github.com/dolthub/doltgresql)

## Description

A version-controlled PostgreSQL-compatible database. `doltgres` is "Dolt for Postgres" — it provides a PostgreSQL-compatible interface to a Dolt database, enabling Git-like version control features (branches, merges, diffs) via standard SQL commands.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/dolthub/doltgresql/releases) and placed in `/usr/local/bin`.
