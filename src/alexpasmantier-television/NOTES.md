# television (`tv`)

## Project

- [television (`tv`)](https://github.com/alexpasmantier/television)

## Description

A blazing fast terminal fuzzy finder. `tv` uses a TUI to allow interactive fuzzy-searching over various sources of data — files, git history, environment variables, and more.

## Installation Method

Downloaded as a pre-compiled binary from the [GitHub releases page](https://github.com/alexpasmantier/television/releases) and placed in `/usr/local/bin`.
